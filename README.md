# Online Shopping Website

A full-stack online shopping website built using Node.js, Express.js, React, JavaScript, and SQL. This project provides an e-commerce platform with user authentication, product browsing, cart management, and order handling.

## Features

- User authentication and login/signup
- Product listing and product management
- Add to cart functionality
- Order management
- Backend APIs using Node.js and Express.js
- SQL database connectivity
- Simple and responsive user interface

## Tech Stack

- Frontend: React, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: SQL / MySQL

## Project Files

- `server.js` – main server file
- `db.js` – database connection
- `auth.js` – authentication functionality
- `products.js` – product-related routes/functions
- `cart.js` – cart-related routes/functions
- `orders.js` – order-related routes/functions
- `hashPasswords.js` – password hashing utility
